# CPA簿記過去問PDFのゲーム用Excel作成依頼

## 目的

PDF `財務会計論_簿記_cpa_過去問_最終.pdf` を、既存のPDFゲームシステムで読み込めるExcel形式に変換する。

特に、PDF後半にある「過去問」セクションも漏れなく取り込むこと。  
後半の過去問は基本的に `1〜6` の選択肢で答える形式なので、Excel上の正解も `1`〜`6` にする。

## Excel形式

ブックには少なくとも以下の2シートを作る。

### シート1: `問題`

1行目はヘッダー。

| 列 | 内容 |
|---|---|
| A | 問題番号 |
| B | 1問目の回答範囲 |
| C | 1問目の正解 |
| D | 2問目の回答範囲 |
| E | 2問目の正解 |
| ... | 同じ形式で最大25問目まで |

最大25小問まで対応する。  
つまり、`回答範囲` と `正解` のペアを25組作る。

### シート2: `ページ情報`

1行目はヘッダー。

| 列 | 内容 |
|---|---|
| A | 問題番号 |
| B | 表示する問題の開始ページ |
| C | 表示する問題の終了ページ |
| D | 解答・解説の開始ページ |
| E | 解答・解説の終了ページ |

PDFページ番号は、PDFビューア上のページではなく、PDFファイル内のページ番号として扱う。

## 問題の分け方

### 前半〜中盤の個別問題・大問形式

`問題1-1`, `問題1-2` のように、同じ大問番号の小問は同じExcel行にまとめる。

例:

- `問題1-1`
- `問題1-2`
- `問題1-3`

これらは1行にまとめる。

ただし、`問題2-1` になったら次のExcel行にする。  
つまり、`問題1-x` と `問題2-x` を同じ行に混ぜない。

### 後半の過去問形式

PDF後半には、以下のような形式の過去問がある。

```text
【令和５年第Ⅰ回 問題５】
...
1 ...
2 ...
3 ...
4 ...
5 ...
6 ...
```

解答解説側には次のような形式で正解が載る。

```text
正解 ５
```

この後半過去問は、1問につきExcel 1行にする。  
回答範囲は問題タイトルが分かる形にしてよい。

例:

| 問題番号 | 1問目の回答範囲 | 1問目の正解 |
|---|---|---|
| CPA短答過去問001_Q05 | 令和５年第Ⅰ回 問題５ | 5 |

## 重要な注意点

### 1. 後半過去問を漏らさない

以前の作成では、PDF後半の過去問セクションがExcelに入っていなかった。  
これは不備なので、必ず後半まで確認する。

目安として、PDFは約975ページあり、後半過去問は700ページ台以降にも存在する。

### 2. 正解の数字は全角にも対応する

PDF本文では、正解が半角 `1`〜`6` ではなく、全角 `１`〜`６` で抽出される場合がある。

そのため、抽出時は以下を同一視する。

| PDF上の文字 | Excelに入れる値 |
|---|---|
| １ | 1 |
| ２ | 2 |
| ３ | 3 |
| ４ | 4 |
| ５ | 5 |
| ６ | 6 |

### 3. 複数問題が同じPDFページにある場合

後半過去問では、1ページ内に複数の問題番号がある場合がある。  
その場合でも、問題ごとにExcel行を分ける。

例:

```text
問題23
問題24
問題25
問題26
```

これらが同じPDFページ内にあっても、Excelでは4行に分ける。

### 4. 解答ページも同様に複数正解がある場合がある

解答解説ページ側でも、1ページに複数の `正解 １〜６` が載っていることがある。  
問題マーカーの出現順と、解答マーカーの出現順を対応させる。

## 期待する検証結果

作成後、最低限以下を確認する。

- `問題` シートと `ページ情報` シートの行数が一致している
- すべての問題番号にページ情報がある
- PDFページ指定が1〜PDF総ページ数の範囲内にある
- 1行に26問以上の小問が入っていない
- 後半過去問の正解がすべて `1`〜`6` のどれかになっている
- 後半過去問が100問程度入っている

参考までに、こちらで作った版では以下だった。

```text

後半短答過去問: 104問
合計: 256行
PDF最大参照ページ: 973
ページ情報欠落: 0

25小問超過: 0
後半過去問の不正な解答: 0
```

## 原因メモ

「過去問なのに1〜6選択にならなかった」原因は、最初のExcel生成ロジックがPDF前半〜中盤の個別問題・大問形式だけを対象にしており、PDF後半の過去問形式を対象にしていなかったため。

さらに、過去問の正解が全角数字 `１〜６` で抽出されることがあり、半角数字だけを見ていると正解を拾えない。

したがって、後半過去問用の抽出処理では以下が必要。

- `【令和... 問題...】` のような問題マーカーを拾う
- `正解 １〜６` または `正解 1〜6` を拾う
- 全角数字を半角数字に正規化する
- 同一ページ内の複数問題・複数解答にも対応する
